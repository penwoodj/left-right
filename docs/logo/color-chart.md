# Left-Right Color System

## Derived Averages

These are the computed averages from each color family, used throughout the project.

### Averaged Green — `#18886c`
- **RGB:** (24, 136, 108)
- **HSL:** 166°, 70%, 31%
- **Usage:** Logo `_<` group, left gradient primary
- **Role:** Represents the "left" / input direction in Left-Right notation

### Averaged Purple — `#b358b5`
- **RGB:** (179, 88, 181)
- **HSL:** 299°, 38%, 53%
- **Usage:** Logo `_>` group, right gradient primary
- **Role:** Represents the "right" / output direction in Left-Right notation

### Averaged Blue (with dark tinge) — `#3d71bf`
- **RGB:** (61, 113, 191)
- **HSL:** 216°, 54%, 49%
- **Usage:** Logo `{ }` brackets, structural elements, headings
- **Role:** Represents the structural/containment aspect of Left-Right notation
